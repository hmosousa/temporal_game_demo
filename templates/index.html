Guidelines:

  • Drag and drop the ">", "<", "=", or "-" symbols to fill the cells.
  • Double-click a filled cell to empty it.
  • Use ">" if the row entity happens after the column entity.
  • Use "<" if the row entity happens before the column entity.
  • Use "=" if the row entity and column entity happen at the same time.
  • Use "-" if the entities are not related.
  • Click the "Reset" button to clear all cells.
